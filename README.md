# Blockchain Explorer Mini

This project helps beginners learn how blockchain explorers work.

You will learn:
- How to connect to a blockchain RPC
- How to read block data
- How to read wallet balances
- How to show blockchain info on a webpage
- How Web3 apps fetch on-chain data

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/blockchain-explorer-mini.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- explorer.js = Blockchain read functions
- web3.js = Blockchain connection
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- logger.js = Logs
- .env.example = Environment example

## Features
- Search block number
- Show block info
- Search wallet address
- Show wallet balance
- Simple blockchain explorer UI

## Learning Goal
This project explains how block explorers read live blockchain data.
